# 芯师爷官网 — 产品需求规格说明书 (PRD)

## 一、产品概述

### 1.1 产品定位

芯师爷官网是一个面向中国半导体产业的 **专业垂直媒体+企业服务平台**，同时面向公众、入驻企业和平台运营三方提供服务。

### 1.2 用户角色

| 角色 | 描述 | 核心需求 |
|------|------|---------|
| **访客** | 未登录的行业从业者 | 浏览内容、搜索信息 |
| **注册用户** | 已注册的个人用户 | 收藏内容、参与投票、投递简历 |
| **入驻企业** | 已认证的半导体企业 | 管理品牌主页、发布内容、投放广告 |
| **运营编辑** | 芯师爷编辑团队 | 内容管理、企业审核、数据管理 |
| **平台管理员** | 芯师爷系统管理员 | 全局配置、用户管理、定价策略 |

## 三、非功能需求

### 3.1 性能需求

| 指标 | 目标 |
|------|------|
| 首屏加载时间 | < 1.5s (LCP) |
| 页面交互响应 | < 100ms |
| API 响应时间（读） | < 200ms (P95) |
| API 响应时间（写） | < 500ms (P95) |
| 同时在线用户 | 支持 5000+ 并发 |
| 可用性 | 99.9% |

### 3.2 安全需求

| 需求 | 要求 |
|------|------|
| 数据传输 | 全站 HTTPS |
| 认证方式 | JWT Token（过期时间策略） |
| 密码存储 | bcryptjs 加密 |
| XSS 防御 | 输入输出编码 |
| CSRF 防御 | Token + SameSite Cookie |
| SQL 注入 | Prisma 参数化查询 |
| 速率限制 | Redis 限流中间件 |
| 文件上传 | 类型校验 + 大小限制 |

### 3.3 可用性需求

- 响应式设计：完美适配桌面端（1440px+）和平板端（768px+）
- 浏览器兼容：Chrome / Firefox / Safari / Edge 最新两个大版本
- 无障碍支持：WCAG 2.1 AA 标准
- 离线降级：核心页面支持 Service Worker 缓存

### 3.4 SEO 需求

- 所有前台页面 SSR 渲染，支持搜索引擎爬取
- 动态生成 `<title>` / `<meta>` / `<meta property="og:">` / `<meta name="twitter:">`
- 自动生成 sitemap.xml
- 结构化数据标注（JSON-LD）
- 文章详情页支持 canonical URL

### 3.5 国际化需求

- 支持中英文双语
- URL 结构：`/zh/...` 和 `/en/...`
- 内容管理后台支持按语言编辑
- 日期、数字格式按语言本地化

**P0（必须）**：首页、认证系统、企业展示、文章系统、搜索、关于我们  
**P1（重要）**：活动管理、评选系统、广告系统、招聘系统、报告系统、国际化、企业后台、管理后台
